Emulator :

Description :
Emulateur d'un MMORPG, codé en Java, utilisant des bases de données. L'émulateur agit comme un serveur et reprend les "règles" du jeu (les formules de calcul, la logique générale). Il utilise des packets pour dialoguer avec l'interface du Jeu.

Mode de réalisation : En équipe.
Langage : Java + SQL.
Période : 2012 - 2018.
